# MP3 Tag Reader

## Overview
A command-line application developed in C to read and display MP3 metadata using ID3 tag extraction techniques.

## Features
- Extract song title
- Display artist details
- Read album information
- Display genre and year
- Binary file processing

## Technologies Used
- C Programming
- File Handling
- ID3 Tag Parsing
- String Manipulation

## Learning Outcomes
- Understanding MP3 file structures
- Binary file handling
- Metadata extraction techniques
- Byte-level data processing
